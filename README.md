# 🚀 Smart To-Do Planner App

A modern, responsive To-Do Planner web app built using HTML, CSS, and JavaScript.

---

## 🌟 Features

- 📅 Date-based task management
- 📝 Add notes for each task
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Data stored using localStorage
- ⌨️ Add task using Enter key
- 🎨 Modern dark UI (Glassmorphism + Blue theme)
